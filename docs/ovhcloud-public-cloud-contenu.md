### Avantages

- **Flexibilité et évolutivité** : Automatisez votre infrastructure avec nos services de conteneurs et d'orchestration managés. Ajoutez et supprimez des ressources directement depuis votre espace client ou via l'API.

- **Coûts prédictibles et facturation transparente** : Payez uniquement pour ce que vous consommez. Les appels API et le trafic réseau sont inclus. Si vous souhaitez vous engager dans la durée, les Saving plans vous feront faire des économies !

- **Interopérabilité et standardisation** : Offrez-vous la liberté d'hybrider votre cloud selon vos besoins. Basé sur des standards open source comme OpenStack, Kubernetes ou PostgreSQL, notre cloud est interopérable et vous permet d'éviter le vendor-lockin.

- **Résilience pour vos applications critiques** : Nous travaillons chaque jour pour vous fournir les meilleurs standards en matière de haute disponibilité et de durabilité sur nos solutions cloud déployées en région 1-AZ ou 3-AZ. Et ce, toujours au meilleur rapport performance-prix !

- **Localisation de vos données** : Localisez vos données au plus près de vos utilisatrices et utilisateurs avec nos déploiements dans les Local Zones. 16 Local Zones déjà disponibles. Notre projet : 150 Local Zones pour vous rapprocher de votre clientèle.

- **Souveraineté et sécurité de vos données** : Partout où vous les déployez, vos données restent vos données. Nos services cloud sont pour la plupart déjà certifiés et pour les autres en cours de certification ISO/IEC 27001, ISO/IEC 27017, ISO/IEC 27018, ISO/IEC 27701, SOC 2 Type II, CSA Type II et C5 Type II.

## Compute (Instances)
**URL:** https://www.ovhcloud.com/fr/public-cloud/instances/

### La puissance des machines virtuelles pour tous vos usages

Bénéficiez de la fiabilité, de la flexibilité et de la réversibilité du cloud pour tous vos projets de modernisation d'applications et de déploiements cloud native.

### Gamme d'offres

**Virtual Machine Instances**
- Instances polyvalentes adaptées à tous usages
- Facturation à l'heure sans engagement
- API complète, Terraform, SDK

**Cloud GPU**
- Jusqu'à 1 000x plus rapides pour calculs parallèles
- GPU NVIDIA A10, A100, H100
- Optimisé pour l'IA et le machine learning

**Metal Instances**
- Combinaison serveurs dédiés + flexibilité cloud
- Performance maximale
- Automatisation API

### Caractéristiques

- Facturation à l'heure, mensuelle ou avec engagement
- Automatisation complète via API, Terraform et SDK
- Disponibilité mondiale via Local Zones et régions 1-AZ ou 3-AZ
- Ratio prix-performance concurrentiel
- Green technology avec système de watercooling
- 20+ ans d'expertise cloud

### Services complémentaires

- **Instance Backup** : Sauvegardes système à la demande avec triple réplication
- **Private Image Catalog** : Systèmes d'exploitation paramétrés
